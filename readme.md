# Google Search Console Plugin

A plugin for [Kirby 2 CMS](http://getkirby.com) that generates an Google Search Console confirmation page.

## Installation

Copy or link the `google` directory to `site/plugins/` **or** use the [Kirby CLI](https://github.com/getkirby/cli):

```bash
kirby plugin:install schnti/kirby-google
```

### Config Variables

```php
c::set('ka.google.verificationCode', 'google9d...e8.html');
```

## How to use it

### google9d...e8.html
Visit the confirmation page at this url: http://example.com/google9d...e8.html.

There is no actual file generated.
